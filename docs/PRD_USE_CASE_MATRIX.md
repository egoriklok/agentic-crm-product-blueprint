# PRD Use Case Matrix

| Persona | Job To Be Done | CRM Module | Generated Output | Acceptance Criteria |
| --- | --- | --- | --- | --- |
| Founder | Understand market and launch sales quickly | Company intelligence, pipeline | CRM workspace and launch plan | Sources registered, leads segmented, next actions visible |
| Sales manager | Work opportunities without losing context | Accounts, contacts, deals | Pipeline, scripts, tasks | Every deal has stage, value, contact and next action |
| Product manager | Present products clearly | Product catalog | Client portal/catalog | Products have descriptions, source references and media |
| Operations manager | Fulfill orders or services | Orders, integrations | Order workflow and export | Orders are auditable and exportable |
| AI agent | Maintain data and workflows | Agent console, approval queue | Tasks, runs, memories | Agent actions are logged and reviewable |
